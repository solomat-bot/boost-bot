# 🚀 BUST | Финансовый навигатор для селлеров

**БУСТ** помогает владельцам онлайн-магазинов на маркетплейсах находить точки роста, управлять прибылью и понимать свои цифры.

---

### 🌿 Возможности

- Экспресс-диагностика прибыли
- Подсказки и советы от финансового аналитика
- Интерактивный ассистент прямо в Telegram
- Простая структура: главная, услуги, ясность и советы

---

### 💡 Технологии

HTML5, CSS3, JavaScript (Vanilla)  
SVG-иконки и адаптивный интерфейс
